Instrucciones de configuración:
--------------------------------
1. Asegúrate de tener un servidor web y un sistema de gestión de bases de datos MySQL (como XAMPP o WAMP) instalados en tu computadora.
2. Importa el archivo "wordpress.sql" en tu sistema de gestión de bases de datos para crear la base de datos y las tablas necesarias.
3. Verifica que el archivo "mapa.php" exista.

Instrucciones de instalación:
------------------------------
1. Descarga todos los archivos proporcionados en este proyecto y colócalos en la carpeta raíz de tu servidor web local.
2. Configura el servidor web y la base de datos siguiendo las instrucciones de configuración anteriores.
3. Accede a la URL "localhost\wordpress" en tu navegador para ver el plugin en uso.

Manifiesto de archivos:
-------------------------
- mapa.php: documento donde se encuentra el codigo usado para el plugin
- docs: Carpeta donde se encuentra el manual creado por phpDocumentor.
